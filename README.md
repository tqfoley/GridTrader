# GridTrader
Schwab API OAuth 2.0 example for grid trading

The majority of this was written with Claude, bugs and errors might exist, use at your own risk

This requires Schwab API access via https://developer.schwab.com/ and a seperate Schwab Developers Portal account to create API keys

OAuth 2.0 requires a https URL for the callback URL specified in appsettings.json

Localy serves to http://localhost:5000/Schwab/TotpSetup
